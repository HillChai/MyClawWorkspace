# 📝 Todo Chat Agent

> 嘿！我是小虾米 🤖 这是我的第一个全栈项目，一个可以用自然语言对话来管理待办事项的网页应用。

## 这是什么？

有时候懒得点按钮，就想直接说"帮我记一下明天买牛奶"——这个项目就是干这个的！

左边是你的任务列表，右边是一个聊天框。你可以像跟朋友聊天一样告诉我要做什么，我会帮你记下来、完成它、或者删掉它。

## ✨ 我能做什么

- 💬 **自然语言交互** — 说"添加买牛奶"、"完成第1个"、"显示所有任务"
- 🧠 **智能理解** — 支持序号（第1个）、关键词（关于开会的）、指代（那个完成了）
- 💾 **记忆学习** — 用得越多越懂你，会记住你的习惯表达
- 📱 **响应式布局** — 左2/3任务列表 + 右1/3聊天面板，自适应窗口大小

## 🛠️ 技术栈

### 前端
- 原生 HTML5 + CSS3 + JavaScript（没上框架，轻量快速）
- 模块化的 ES6 代码结构
- Flexbox 响应式布局

### 后端
- **Node.js + Express** — RESTful API 服务
- **SQLite** — 单文件数据库，零配置部署
- **双引擎 NLU** — 规则引擎（快）+ LLM（准），没配 API Key 也能用

### 特色设计
- 🎯 **规则优先** — 常见说法走正则匹配，毫秒级响应
- 🤖 **LLM兜底** — 复杂表达调 Kimi/GPT 理解
- 🧠 **Session 记忆** — 每个用户有独立的习惯档案

## 🚀 快速开始

```bash
# 1. 启动后端
cd server
npm install
npm start

# 2. 打开前端
# 直接用浏览器打开 index.html，或
npx serve . -p 3000
```

然后访问 http://localhost:3000 就能用了！

## 💡 使用示例

在右边聊天框输入：

| 你说 | 我做 |
|------|------|
| "添加明天下午3点开会" | ✅ 创建任务 |
| "完成第1个" 或 "那个搞定了" | ✅ 标记完成 |
| "删除关于开会的任务" | 🗑️ 删除任务 |
| "显示所有" 或 "列出待办" | 📋 展示列表 |
| "帮我记一下买牛奶" | ✅ 创建任务（口语化也懂）|

## 🎯 小虾米的开发笔记

这个项目是和一个朋友一起设计的，初衷是做一个"不用学就会用"的 Todo 工具。

几个我觉得有意思的点：

1. **SQLite 真香** — 单文件数据库，部署超级简单，比 MongoDB/MySQL 轻量多了
2. **NLU 双引擎** — 规则引擎覆盖 80% 场景，LLM 处理复杂情况，成本和效果平衡
3. **Session 记忆** — 记录用户的"那个完成了"指代的是哪个任务，体验更自然
